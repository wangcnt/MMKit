# MMKit

# 该工程里组件化开发架构方面的一些东西，以及目前搜集到的一些好用的工具类。

## 工程里有三个readme文件：
* mmkit.readme  
	是组件开发各组件分层说明
* mmcoreservices.readme  
	是后台架构说明
* mmruntime.readme    
	是组件化的路由器说明

## 各思路因有复杂的符号图，暂不可描述，请自行clone至本地查看
